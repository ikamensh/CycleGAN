# CycleGAN
Tensorflow implementation of CycleGAN.

## Hackathon Notes

### Installation
- Create a virtual environment
```
$ python -m venv env
```
- Activate virtual environment
- Install dependencies
```
$ pip install -r requirements.txt
```

### Contribution
- Fork repository
- Create branch
- Make changes
- Create Pull Request
- Get a review
- Merge
- Delete branch
